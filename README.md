# My Webpage

This repository contains the HTML files for my webpage.


## Files

- A basic index html file showing a basic appearance of a HTML file.
- `index.html`: The main page of the website. Contains a header with links to the other pages, a main section with the content of the page, and a footer with information about the author.
- `tweets.html`: A page that displays tweets. Contains a header with links to the other pages, a main section with the content of the page, and a footer with information about the author.

## Structure

The structure of the HTML files is as follows:

- The `DOCTYPE` declaration specifies that the file is an HTML5 document.
- The `head` element contains metadata about the page, including the character encoding and the title of the page.
- The `body` element contains the content of the page, organized into `header`, `main`, and `footer` elements.
- The `header` element contains an unordered list with links to the other pages of the website.
- The `main` element contains the content of the page, organized into `article` and `aside` elements.
- The `article` element contains the main content of the page.
- The `aside` element contains additional information or content related to the main content.
- The `footer` element contains information about the author of the page.

## Usage

To view the website, open the `index.html` file in a web browser. You can navigate to the other pages of the website by clicking on the links in the header.
